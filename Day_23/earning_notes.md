# Backend Development – Day 23 Notes

## What is Deployment?

Deployment is the process of making an application available on the internet so that users can access it.

## Why Deployment is Important?

- Makes the application accessible globally
- Allows real users to interact with the system
- Essential step after development

## Basic Deployment Steps

1. Prepare the application
2. Configure environment variables
3. Choose a hosting platform
4. Upload code to server
5. Run the application

## Types of Servers

Local Server
Runs only on your system

Remote Server
Hosted on the internet (cloud platforms)

## Environment Configuration

Environment variables are used to:
- Store sensitive data (API keys, DB URLs)
- Configure application settings
