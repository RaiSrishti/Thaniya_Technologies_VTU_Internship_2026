# Backend Development – Day 16 Notes

## What is API Security?

API security refers to protecting backend endpoints from unauthorized access or malicious attacks.

## Common Security Practices

Authentication
Authorization
Input validation
Rate limiting
Secure environment variables

## Authentication vs Authorization

Authentication
Verifies the identity of the user.

Authorization
Determines what actions the user is allowed to perform.

## Basic Authentication Flow

Client sends credentials → Server verifies credentials → Access granted or denied.

## Importance of API Security

- Protects sensitive data
- Prevents unauthorized access
- Maintains application integrity

## Learning Outcome

Today I learned the importance of securing backend APIs and how authentication mechanisms help protect application resources.
