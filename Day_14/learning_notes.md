# Backend Development – Day 14 Notes

## Why Databases are Important

Databases allow applications to store and manage large amounts of data efficiently.

Examples:
User accounts
Product information
Application logs

## Types of Databases

Relational Databases (SQL)
Examples: MySQL, PostgreSQL

Non-relational Databases (NoSQL)
Examples: MongoDB, Firebase

## Backend and Database Interaction

Client Request → API Route → Database Query → Response Sent

Example Operations

Create Data – Insert new records  
Read Data – Retrieve stored information  
Update Data – Modify existing records  
Delete Data – Remove records

## Database Integration Steps

1. Install database driver or ORM
2. Establish database connection
3. Define schema or table structure
4. Perform CRUD operations through API

## Learning Outcome

Today I learned how backend APIs interact with databases to store and retrieve data dynamically. Database integration is a key part of building scalable and data-driven backend applications.
