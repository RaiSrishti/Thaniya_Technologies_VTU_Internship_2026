# Day 20 – Mini Backend Project: User Management API

On Day 20, I implemented a mini backend project by combining all the concepts learned during the internship. I developed a User Management API using Node.js, Express.js, and MongoDB.

The project includes features such as creating users, retrieving user data, updating user details, and deleting users. It also follows a structured MVC architecture to organize the code efficiently.

This project helped me understand how real-world backend systems are designed and developed by integrating APIs, database operations, and modular architecture.
