# Backend Development – Day 20 Notes

## Project Overview

A User Management System that performs CRUD operations using REST APIs and MongoDB.

## Features Implemented

- Create User
- Get All Users
- Update User
- Delete User

## Technologies Used

- Node.js
- Express.js
- MongoDB
- Mongoose

## Project Structure (MVC)

models → Database schema  
controllers → Business logic  
routes → API endpoints  
server.js → Entry point  

## Key Learning Outcome

Today I applied all backend concepts into a single project. This helped me understand how APIs, database, and project structure work together to build a complete backend system.
