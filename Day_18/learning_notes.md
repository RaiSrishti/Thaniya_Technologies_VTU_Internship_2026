# Backend Development – Day 18 Notes

## What is API Testing?

API testing ensures that backend endpoints work correctly and return the expected responses.

Developers test APIs by sending HTTP requests and analyzing the responses.

## Common HTTP Methods Tested

GET – Retrieve data  
POST – Create new data  
PUT – Update data  
DELETE – Remove data  

## Data Validation

Validation ensures that only valid and correct data is accepted by the server.

Examples:
- Checking required fields
- Validating email format
- Preventing empty inputs

## Tools for API Testing

Postman  
Insomnia  
Thunder Client (VS Code Extension)

## Learning Outcome

Today I learned how API testing tools help verify backend functionality and how validation improves data reliability in applications.
